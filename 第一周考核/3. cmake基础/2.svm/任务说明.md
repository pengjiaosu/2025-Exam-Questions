* 如果这是你在本大题中做的第一道题，请返回上一级目录并查看是否有 [**1. cmake_practice**](../1.cmake_practice/%E4%BB%BB%E5%8A%A1%E8%AF%B4%E6%98%8E.md) 文件夹，如果没有请向考核人员反映；

* 本题共**2小问**，请大家认真审题，仔细作答，并在CMakeLists.txt文件中**加入注释用以说明CMakeLists.txt的编写思路**；
* 最终请将第一问截图和带有CMakeLists.txt的完整项目打包提交。


### 1. 编写CMakeLists.txt文件构建SVM工程

* 本小问中不需要使用./test文件夹下的文件，大家可以不用关心这个目录。
* 需要实现的要求：
  * 将./src和./include下的文件编译成静态库和动态库文件，分别命名为libsvm.a和libsvm.so；
  * 将上述两个库文件编译到**./${your_build_dir}/lib**目录下；
  * 将libsvm.so安装到**/usr/local/lib**目录下，将./include/svm文件夹安装到**/usr/local/include**目录下，并截图提交。

### 2. 修改CMakeLists.txt文件，条件编译svm_test项目

* 设置条件编译变量，默认时不将./test文件夹下的文件编译为**svm_test**项目。当输入命令

  ```
  cmake -DBUILD_SVM_TEST=ON ..
  ```

  时，将./test下的main.cpp编译为**svm_test**项目；

* 本题需要用到./data文件夹中的数据，请在编译时将它移动到**./${your_build_dir}/data**目录下。
