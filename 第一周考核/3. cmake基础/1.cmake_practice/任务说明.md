# 第一题：CMake Practice

### 0. 说明

* 本题是 **CMake** 部分的第一题。ROS2的编译是通过colcon实现的，而其底层依旧是CMake。本题主要目的是帮助大家熟悉CMake工具的使用以及CMake的基本命令，请大家事先根据网上的教程安装好CMake；
* 本题一共有 **4小问** ；
* 对考题描述有疑问者请在考核群中向考核人员反映，以考核人员解释为准；
* 感谢Akagi201和browny提供的cmake教程以及练习项目；
* 可以在[这里](https://github.com/Akagi201/learning-cmake/tree/master/docs)找到cmake的pdf版教程，推荐大家阅读一下；
* 本题主要是熟悉cmake基础语法和项目构建方式，也有详细的教学项目和资料，故不用提交代码文件，但cmake部分第二题将要利用本题学到的内容自己编写CMakeLists.txt文件并提交**结果截图**，所以还需认真学习。

### 1. 从git上向本文件夹中下载cmake-practice项目、学习并编译其中的hello_world项目

* 仓库地址(github)：[cmake-practice@github](https://github.com/browny/cmake-practice.git)；
* 仓库地址(gitee(个人拷贝))：[cmake-practice@gitee](https://gitee.com/blusque/cmake-practice.git)；
* 下载后如图所示：![](./2022-07-06%2022-39-29%20的屏幕截图.png)
* 打开cmake-practice，学习并编译第一个项目(hello_world)；
* 录制编译时的cmake输出以及运行编译后得到的项目的输出，截图结果文件，命名为q1。

### 2. 学习并编译cmake-practice中的hello_world_clear项目

* 这个项目将学习如何将文件和编译后的可执行文件下载到系统中；
* 请学习并编译此项目，并截图结果文件(可以做前后对比)，命名为q2；
* 尝试更改CMakeLists.txt文件或增加编译条件，使其下载目录为同文件夹(./cmake-practice/hello_world_clear/)下的./install文件夹，并截图，命名为q3。

### 3. 学习并编译cmake-practice中的hello_world_lib项目

* 这个项目将学习如何编译库文件(包括静态库和动态库)；
* 请学习并编译此项目，无需提交截图。

### 4. 学习并编译cmake-practice中的hello_world_share项目

* 这个项目将学习如何向你的项目链接系统中的静态库；
* 请学习并编译此项目，无需提交截图。

### 最后：

* 这道小题已经结束，但你仍然可以对cmake-practice中的项目或是你自己新建的项目进行更改，并在此过程中学习cmake的使用。学习的过程可以参考[cmake-practice.pdf](https://github.com/Akagi201/learning-cmake/blob/master/docs/cmake-practice.pdf)，这个文档可以满足之后在机器人队的工作需要，你可以保存它并在之后的学习和工作中翻出来看看。
