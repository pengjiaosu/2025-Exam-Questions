# 图搜索

### 任务目标
实现图搜索的经典算法。

### 涉及知识
1. A算法
2. A*算法（包括迪杰斯特拉算法）

### 任务要求
1. *map.png*为地图，规定最左上方的格子为(0,0)，向右为x轴正方向，向下为y轴正方向；
2. 可以朝八个方向运动，不可走出地图，不可走到黑色障碍物上。水平/竖直走一步的路程为1，斜着走一步的路程为根号2；
3. 请按照下面三种启发函数编写启发式搜索算法：0，L1范数，无穷范数；
4. 将地图写到程序里（可以借助opencv读取格子颜色来节约时间），程序输入为起点和终点，输出为分别按照三种启发函数实现的路径，以及体现三种搜索方式效率的指标（例如搜索的轮数，搜索过的点数，搜索花费的时间）；
5. 如有兴趣，可以用自己喜欢的方式将搜索的过程可视化出来。

**注1：请合理调库，核心步骤需自己完成，可以参考互联网资料但不允许照搬他人代码，写好注释表明思路**

**注2：请思考启发函数种类和解路径是否最短之间的关系**

**注3：编程语言限定为c/c++，python**

### 提交的内容

1. 程序源代码
2. 一个md文件，包括：
- 程序说明
- 结果展示
- 对启发式搜索算法的认识
- 分析三种实现方式的计算效率
- 分析得出最短路径的条件
- 其他想法
